SelectionJS Documentation

A JavaScript library for handling text selection and modifications.

Installation

<script src="selection.js"></script>

Configuration

You can configure SelectionJS with the following options:

SelectionJS.configure({
    container: [document.getElementById("editor1"), document.getElementById("editor2")],
    wrapperPlaceholder: "$"
});

Usage

Example of modifying selected text:

SelectionJS.on("select", (context) => {
    context.modify({ action: "wrap", text: "**Bold**", wrapper: "$" });
});
